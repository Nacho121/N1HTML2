Aprendiendo HTML

HTML es el lenguaje basico para crear paginas web

¿Que es HTML

HTML (HyperText Markup Language) es un lenguaje de marcado utilizado para estructurar y presentar contenido en páginas web. No es un lenguaje de programación, ya que no permite realizar cálculos ni ejecutar lógica, sino que define la estructura de los elementos mediante etiquetas. Estas etiquetas indican al navegador cómo debe interpretar y mostrar el contenido al usuario.

Historia del HTML

HTML (HyperText Markup Language) fue creado por Tim Berners-Lee en 1989-1991 en el CERN para compartir documentos científicos mediante hipervínculos. Evolucionó de ser un lenguaje simple con 18 etiquetas a convertirse en el estándar de la web, pasando por versiones clave como HTML 2.0, 3.2, 4.01, hasta el actual HTML5 (2014), que introdujo soporte multimedia y semántica avanzada.

HTML5

HTML5 es la version mas reciente y permite agregar videos, audio y nuevas funcionalidades
